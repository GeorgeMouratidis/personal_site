Database Project

Project Description

In this project, we designed and implemented a database for a bus company based on the itineraries and schedules of the city of Patras in Greece. We first created the Entity Relationship Diagram (ERD), which then got interpreted into a relational schema, in order to create the database from scratch. Afterwards, using Python's library SQlite, we created the relevant tables, and inserted the data of the afforementioned itineraries.

Furthermore, we identified key questions and tasks crucial for both the company and passengers. These ranged from employee management for the company, to providing clients with information on the last schedule of an itinerary. To address these needs, we formulated complex SQL queries and implemented them effectively. To enhance user interaction and visualization of the queries, we developed a user-friendly GUI using Python's Tkinter. This not only facilitates a more comfortable user experience but also streamlines the fulfillment of their requirements.
